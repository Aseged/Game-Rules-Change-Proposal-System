How to create database and start the application.

1. In Netbeans JavaDB create a schema with name, username and password "WOLDESELASSIEA"

2. Open the newly created schema to get to the tables folder. Right click on the tables and choose 'Execute Command'

3. Open and copy all the sql script provided in /Web/db folder.

4. Paste the script in the Command window and click on execute. This will create the tables and populate them with initial data.

5. Run the application.
